LearnGraphQL

Dear diary,

Today I discovered a couple of days ago LearnGraphQL course by @arunoda, and sinceresly, it's awesome.
I had any experience with GraphQL and after following it I managed to build my small API on a MongoDB database.
The only downfall about it is that it later uses promised-mongo to interact with MongoDB API and the steps they provide don't work (at least in my case). So I had to search somewhere else and I ended up using Mongoose.
Except from that everything is very well explained and this small problem is just related with how you fetch data in your program, not GraphQL itself.

Week is ending and I'm hungry to learn more!
